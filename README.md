## First Come, First Serve (FCFS/FIFO) 
This algorithm is used for batch processes, and the output of this code will be like this :

```
Enter number of processes you want (max 20): 5
===============================================

Enter burst times: 
P[1]: 2
P[2]: 3
P[3]: 4
P[4]: 9
P[5]: 52

==============================================

Process	CBT	WT	TaT
P[1]	2	0	2
P[2]	3	2	5
P[3]	4	5	9
P[4]	9	9	18
P[5]	52	18	70
```
